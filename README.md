# [Better Every Day](https://padi.github.io)

A personal blog of Marc Rendl, a software developer.

## Requirements

From: https://jekyllrb.com/docs/installation/#requirements

    Ruby version 2.7.0 or higher, including all development headers (check your Ruby version using ruby -v)
    RubyGems (check your Gems version using gem -v)
    GCC and Make (check versions using gcc -v,g++ -v, and make -v)

## How to Run Locally

```shell
# Install dependencies after you've installation
# the requirements/dependencies above
$ bundle install

# visit page at localhost:4000
$ bundle exec jekyll serve

# if you want to automatically reload the page when you change the source:
$ bundle exec jekyll serve
```
